### Board setup

- Silabs BRD2703A (MG24 explorer kit)
- Pinout:
  - PC00: Red LED
  - PC03: Yellow LED
  - PC08: Green LED
  - PC02: Red Button
  - PC01: Yellow Button
  - PB01: Green Button
  - PA00: Latch 1
  - PB04: Latch 2
  - PB05: Latch 3
  - PB00: 10k pot centertap, powered 3.3V
  - PD04: Proximity sensor
  - PD05: Debug pin
